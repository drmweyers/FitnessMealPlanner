## Executive Summary

We are building a Russell Brunson-style sales funnel for EvoFitMeals that takes prospects through a value ladder from free lead magnet to $399 enterprise tier. The funnel sells **lifetime access** (one-time payment, no subscriptions) to fitness professionals, trainers, dieticians, and gym owners.

**The Big Idea:** "Pay Once. Plan Forever."

**USP:** EvoFitMeals is the ONLY AI-powered meal planning platform with lifetime access for a single payment. While competitors charge $20-$200/month forever, we charge $199-$399 once. Over 10 years, trainers save up to $11,601.

## Recipe Generation System (Selling Point)

### What Makes It Special

EvoFitMeals uses a **BMAD Multi-Agent AI System** — 8 specialized AI agents that work together to generate, validate, and deliver recipes. This is NOT a simple ChatGPT wrapper.

### The 8 AI Agents (Behind the Scenes)

| Agent | What It Does | Why It Matters to Customers |
|-------|-------------|---------------------------|
| Recipe Concept Agent | Plans diverse, non-repeating recipe ideas | "Your clients never get bored with the same meals" |
| Nutritional Validator | Checks every macro within 5g tolerance | "Every plan hits the exact macro targets you set" |
| Image Generator | Creates unique DALL-E 3 images per recipe | "Beautiful, real-looking food photos on every recipe" |
| Database Orchestrator | Saves and organizes everything | "Your recipe library grows automatically" |
| Image Storage Agent | Optimizes and stores images in the cloud | "Fast-loading images, never lost" |
| Progress Monitor | Tracks generation in real-time | "Watch your recipes being created live" |
| BMAD Coordinator | Orchestrates all agents together | "Restaurant-quality results in seconds" |
| Base Agent | Handles errors and retries | "If something goes wrong, it fixes itself" |

### How to Sell the AI System

**Don't say:** "We use GPT-4 and DALL-E 3"
**Do say:** "Our 8-agent AI system generates personalized meal plans with validated macros, beautiful food photography, and grocery lists — in under 60 seconds."

### Generation Capabilities by Tier

| Capability | Starter | Professional | Enterprise |
|------------|---------|-------------|------------|
| Recipes per generation | 1-25 | 1-50 | 1-100 |
| Monthly plan limit | 50 | 200 | Unlimited |
| Dietary protocols | 8+ | 8+ seasonal | All + specialized |
| Natural language input | Basic | Advanced | Advanced + bulk |
| Macro validation | Standard (10%) | Standard (10%) | Strict mode available |
| Image generation | Standard | Standard | Priority queue |
| Meal types | 4 (B/L/D/S) | 4 + seasonal | 4 + seasonal + custom |
| Prep time filters | Yes | Yes | Yes + batch optimization |
| Ingredient constraints | Max 50 items | Max 50 items | Unlimited |

### Meal Plan Types Available

1. **Standard Daily Plans** — 1-30 days, 1-6 meals/day
2. **Calorie-Targeted** — 800-5,000 kcal daily targets
3. **Macro-Specific** — Exact protein/carb/fat gram targets
4. **Dietary Protocol Plans** — Vegan, keto, gluten-free, paleo, Mediterranean, high-protein, low-carb, dairy-free
5. **Prep-Time Optimized** — Plans with max prep time constraints
6. **Ingredient-Limited** — Plans avoiding specific ingredients
7. **Fitness Goal Plans** — Weight loss, muscle gain, maintenance, endurance
8. **Seasonal Plans** (Professional+) — Recipes using in-season ingredients
9. **Longevity Plans** (Professional+) — Anti-aging, blue-zone inspired
10. **Specialized Protocols** (Enterprise) — Parasite cleanse, custom fasting, therapeutic
